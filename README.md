# Editor
A text editor

# Installation
Download `src/editor`
```javascript
import { Editor } from "/path/to/editor/editor.js";
```
or use a CDN
```javascript
import { Editor } from ".https://cdn.jsdelivr.net/gh/ninjamar/editor@latest/src editor/editor.js";
```
# Usage
```javascript
new Editor(editorElement, options);
```
